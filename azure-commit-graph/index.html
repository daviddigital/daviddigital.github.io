Commit Graph for Azure Repos

Purpose: this Azure DevOps extension, published by Cody Labs, adds a contribution heatmap for Azure Repos — the commit-activity graph GitHub and GitLab have, for Azure DevOps.

It runs client-side inside Azure DevOps using your own session. Commit data is read from your organisation and rendered in your browser; we operate no servers and never receive your data.
